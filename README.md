# Demo SaveTest Test Cases

Репозиторий с демонстрационными тест-кейсами для SaveTest TMS.

## Описание

Этот репозиторий содержит примеры тест-кейсов в формате YAML, которые демонстрируют возможности SaveTest TMS. Вы можете использовать эти примеры как основу для создания собственных тест-кейсов.

## Структура репозитория

```
tests/
├── attach/              # Вложения для тест-кейсов
├── test-case/           # Основные тест-кейсы
│   └── test-suite.yaml
└── common-case/         # Общие тест-кейсы
    └── example-common-case.yaml
docs/
└── wiki_example/        # Примеры документации для Wiki
    └── example.md
```

## Использование

1. Клонируйте репозиторий или используйте его как шаблон
2. Создайте проект в SaveTest TMS
3. Укажите URL этого репозитория при настройке проекта
4. Выполните синхронизацию проекта
5. Тест-кейсы будут автоматически загружены в систему
